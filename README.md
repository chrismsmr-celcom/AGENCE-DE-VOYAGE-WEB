# 🌍 Terra Voyage

Agence de voyage de luxe - Plateforme de réservation de vols, hôtels et packages.

## 🚀 Technologies

- **Backend**: Node.js, Express, Duffel API, Supabase
- **Frontend**: HTML5, Tailwind CSS, Vanilla JS
- **Déploiement**: Render

## 📦 Installation

```bash
# Cloner le projet
git clone https://github.com/chrismsmr-celcom/AGENCE-DE-VOYAGE-WEB.git
cd terra-voyage

# Installer les dépendances
npm install

# Configurer les variables d'environnement
cp .env.example .env
# Éditer .env avec vos clés API

# Démarrer en développement
npm run dev
